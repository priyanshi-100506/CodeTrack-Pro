CodeTrack Pro

A sleek web app to track and level up your coding practice  
Live demo: https://codetrackpro.netlify.app

Features
- Track total problems solved, streaks, and accuracy  
- Browse problem sets by difficulty and category  
- Set daily and weekly goals and claim rewards  
- Responsive dashboard with a clean UI  

Tech Stack
- Frontend: React, Tailwind CSS  
- State: Context API or Redux  
- Backend: Firebase or REST API  
- Hosting: Netlify  

Setup
1. Clone the repository  
   git clone https://github.com/your-username/codetrack-pro.git  
   cd codetrack-pro  

2. Install dependencies  
   npm install  

3. Run the app  
   npm start  

Contribute
